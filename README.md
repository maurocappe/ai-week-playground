# 🎮 AI Week Playground

Try Claude Code from your browser — nothing to install. This template gives you a cloud dev machine with Claude Code already set up.

## Get started (3 steps)

1. Click the green **Use this template** button (top right) → **Open in a codespace**. Wait a minute or two while your machine boots.
2. In the terminal at the bottom, type `claude` and press Enter. Follow the login: click the link, sign in to Claude, **copy the code it shows you, and paste it back into the terminal**.
3. Ask Claude to build something! Try this first:

> Build me a snake game I can play in the browser — arrow keys to move, a score counter, and a game-over screen with a restart button. Make it look slick. Then serve it on port 8000.

When the **"Open in Browser"** notification pops up, click it. That's your game, running live. 🐍

## Then keep going

Just keep talking to Claude:

- "Make the snake rainbow-colored"
- "Make it speed up as my score grows"
- "Add sound effects"
- "Actually, turn this into Tetris"

Refresh the preview tab after each change.

## Good to know

- **⚠️ No internal Anthos data** — this playground is for general experimentation only. Build things from public or made-up data; don't paste in or connect anything proprietary (Lodestone, Affinity, deal/portfolio data, internal docs, company MCPs).
- **It's free** — personal GitHub accounts include ~60 hours/month of codespace time. Machines auto-sleep after 30 minutes idle.
- **It's a sandbox** — this codespace can't touch anything else. Worst case, delete it at [github.com/codespaces](https://github.com/codespaces) and start fresh.
- **Keep your work** — Source Control icon (left sidebar) → **Publish to GitHub** saves it as a repo on your account.
